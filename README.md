# Airplane Reservation System ✈️

A desktop-based flight booking application designed for efficiency and modern aesthetics. This project demonstrates integration between Python GUIs and relational databases.

## 🚀 Key Features
* **Modern UI/UX**: Built with `CustomTkinter` for a sleek, responsive dark-themed dashboard.
* **Real-time Search**: Seamlessly filter and view available domestic flights in India.
* **Booking Cart**: A dedicated sidebar to manage your selections and track your bookings in real-time.
* **Budget Tracker**: Automatic total price calculation with built-in alerts when the budget limit is exceeded.
* **Database Driven**: Uses `MySQL` (via XAMPP) for persistent storage of flight and passenger data.

## 🛠️ Tech Stack
* **Language**: Python 3.x
* **GUI Framework**: CustomTkinter
* **Database**: MySQL (phpMyAdmin)
* **Connectivity**: mysql-connector-python

## ⚙️ Setup Instructions
1. **Clone the repository**: `git clone https://github.com/TanmayAgg890/Airways-Reservation-System.git`
2. **Install dependencies**: `pip install -r requirements.txt`
3. **Database Configuration**:
    * Open XAMPP Control Panel and start **Apache** and **MySQL**.
    * Import the provided SQL database into phpMyAdmin.
    * Ensure your database name matches the one in your script.
4. **Run the application**: `python "Airplane reservaltion edited.py"`
